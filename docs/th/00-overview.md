# Spring Boot 4.1 Features - Overview

## Objective

เป้าหมายของ Repository นี้คืออธิบาย Feature ของ Spring Boot 4.1 จากมุมมองงาน Production และ Software Engineering

## Purpose

Spring Boot มีความสามารถหลายอย่างอยู่แล้ว แต่ Feature ใหม่มักเกิดจากข้อจำกัดหรือ Pain Point ของของเดิม Repository นี้จะอธิบายว่า "ทำไมต้องมีของใหม่" ไม่ใช่แค่ "ของใหม่ใช้ยังไง"

## Expectations

ผู้อ่านจะได้ทั้งแนวคิด, Code ตัวอย่าง, Trade-off และ Use Case จริง แต่ Repository นี้ไม่ได้แทน Official Documentation
